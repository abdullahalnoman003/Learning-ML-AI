##  Learning Philosophy

> *"Understand deeply → Implement simply → Document clearly"*

This repository is built on the belief that the best way to learn is to:
- **Write it from scratch** before using libraries
- **Explain every line** as if teaching a beginner
- **Build progressively** — each phase depends on the last
